# MileMetrics

A simple and intuitive app for tracking your daily mileage.

## 🚀 About

MileMetrics helps you log, track, and review your miles with ease.  
Built with a focus on simplicity and clean user experience. The developer was inspired to begin developing this app after having fustration with other overcomplicated and clunky miletracking apps, and after desiring an app to faciliate the tracking of miles during gigwork like DoorDash for tax purposes.

## ✨ Features

- Log miles quickly
- View daily and total mileage
- Clean, mobile-friendly interface

## 🚧 Current Status

Currently working on improving mobile responsiveness and overall UI polish.

## 🛠️ Tech Stack

- React
- React Router
- Tailwind CSS

- Improve dashboard insights
